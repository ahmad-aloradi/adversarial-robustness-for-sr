# tensorflow 2.18 needs `numpy<2.1.0`. Brevitas's pin is overly conservative —
# it runs fine on numpy 2.0.x — so we install it with --no-deps to bypass it.
# Its only non-stdlib runtime dep is `unfoldnd`, pinned below.
