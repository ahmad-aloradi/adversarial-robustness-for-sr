Create `scripts/average_checkpoints.py` following the algorithm from WeSpeaker's `average_model.py`:

1. **Script structure:**
   - Shebang: `#!/usr/bin/env python`
   - Module docstring with usage example
   - Single function `average_checkpoints(src_dir, output_path, num_avg)`
   - CLI interface via argparse

2. **Core algorithm:**
   ```python
   # Find checkpoint files (Lightning format: epoch=X-step=Y.ckpt or epoch=X-*.ckpt)
   ckpt_files = list(Path(src_dir).glob("epoch=*.ckpt"))

   # Sort by epoch number (ascending)
   def get_epoch(p):
       return int(p.stem.split('epoch=')[1].split('-')[0])
   ckpt_files = sorted(ckpt_files, key=get_epoch)

   # Take last N (skip 'last.ckpt' and 'best.ckpt' symlinks)
   ckpt_files = [f for f in ckpt_files if f.stem.startswith('epoch=')]
   ckpt_files = ckpt_files[-num_avg:]

   # Load and accumulate
   avg_state = None
   for path in ckpt_files:
       checkpoint = torch.load(path, map_location='cpu')
       state = checkpoint['state_dict']
       if avg_state is None:
           avg_state = {k: v.float().clone() for k, v in state.items()}
       else:
           for k in avg_state:
               avg_state[k] += state[k].float()

   # Divide by count
   for k in avg_state:
       avg_state[k] = avg_state[k] / len(ckpt_files)

   # Save in Lightning-compatible format
   output_checkpoint = {
       'state_dict': avg_state,
       'averaged_from': [str(f) for f in ckpt_files],
       'num_averaged': len(ckpt_files),
   }
   torch.save(output_checkpoint, output_path)
   ```

3. **Edge cases to handle:**
   - Fewer than num_avg checkpoints available: warn and average all found
   - No checkpoints found: raise clear error
   - Print which checkpoints are being averaged

4. **CLI interface:**
   ```python
   parser = argparse.ArgumentParser(description="Average PyTorch Lightning checkpoints")
   parser.add_argument("--src_dir", required=True, help="Directory containing checkpoints")
   parser.add_argument("--output_path", required=True, help="Output path for averaged checkpoint")
   parser.add_argument("--num_avg", type=int, default=10, help="Number of checkpoints to average (default: 10)")
   ```

5. **Style requirements:**
   - Follow codebase style (see CLAUDE.md)
   - No excessive error handling - let failures surface naturally
   - Type hints for public function signature
   - Concise docstring explaining what it does

```python
import torch
import tempfile
from pathlib import Path
import subprocess

# Create temp dir
with tempfile.TemporaryDirectory() as tmpdir:
    tmpdir = Path(tmpdir)

    # Create 3 mock checkpoints with known values
    for i in range(3):
        state = {'weight': torch.tensor([float(i + 1)])}  # 1.0, 2.0, 3.0
        torch.save({'state_dict': state}, tmpdir / f"epoch={i:03d}-step=100.ckpt")

    output_path = tmpdir / "avg_model.ckpt"

    # Run script
    result = subprocess.run([
        'python', 'scripts/average_checkpoints.py',
        '--src_dir', str(tmpdir),
        '--output_path', str(output_path),
        '--num_avg', '3'
    ], capture_output=True, text=True)

    print(result.stdout)
    if result.returncode != 0:
        print(result.stderr)
        raise RuntimeError("Script failed")

    # Verify
    avg = torch.load(output_path)
    expected = 2.0  # (1 + 2 + 3) / 3
    actual = avg['state_dict']['weight'].item()
    assert abs(actual - expected) < 1e-6, f"Expected {expected}, got {actual}"
    print(f"Verification passed: averaged value = {actual}")
```
